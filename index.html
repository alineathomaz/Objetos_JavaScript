Nesta aula vamos aprender sobre os objetos JavaScript.

Ele nos permite armazenar os dados em um lugar, que pertence a um determinado objeto. Então, poderemos acessá-los e alterá-los com muita facilidade, o que aprenderemos com esta atividade de programação.

Para definir um objeto, primeiro declaramos um nome de variável e o definimos igual a chaves vazias.

Exemplo:

var nome_do_objeto = { };

No final dos parênteses, haverá um ponto e vírgula. Assim que tivermos nosso objeto vazio pronto, podemos começar a adicionar os dados a ele.

Uma vez que este objeto é denominado, iremos adicionar dados que são relevantes para ele.

Esses dados são armazenados em um par de valores-chave; a chave é o nome que define o valor. O valor é atribuído a uma propriedade/atributo usando dois pontos (:), em vez do símbolo de igual.

Exemplo:

var nome_do_objeto = {

propriedade: valor,

};
